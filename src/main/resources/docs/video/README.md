# video

Recording Chrome-backed Selenium sessions through Chrome DevTools screencast.

- [VideoRecorder](VideoRecorder.md) - starts and stops recordings and applies
  the retention policy.
- [VideoRecordingOptions](VideoRecordingOptions.md) - output folder, retention,
  frame rate, JPEG quality, and maximum frame size.
- [VideoRetentionPolicy](VideoRetentionPolicy.md) - whether recordings are kept
  always or only on failure.
- [VideoRecordingException](VideoRecordingException.md) - runtime exception for
  recording failures.
