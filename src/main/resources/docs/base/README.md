# base

The foundation every page object is built on.

- [BasePage](BasePage.md) — the class every page object extends.
- [ActionsContainer](ActionsContainer.md) — builds one instance of every action group.
- [PageModularOptions](PageModularOptions.md) — settings that customise the action groups.
- [ReaderActionsContainer](ReaderActionsContainer.md) — groups the four document readers together.
