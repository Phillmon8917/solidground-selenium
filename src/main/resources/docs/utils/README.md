# utils

Shared, cross-cutting utilities used by the action classes.

- [env](env/README.md) — reading configuration from `.env`, environment variables, or system properties.
- [logging](logging/README.md) — the shared logger and its pluggable reporters.
- [timeout](timeout/README.md) — adjusting timeouts so CI runs get extra buffer.
