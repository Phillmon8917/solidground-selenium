# modulars/element

The main element actions: click, type, read text/attributes, check
state, scroll, and wait — available on a page object as
`modulars.elementActions`.

- [ElementActions](ElementActions.md) — the full set of element actions.
- [ElementActionException](ElementActionException.md) — thrown when a wait for an element's state times out.
