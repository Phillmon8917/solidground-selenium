# modulars/dropdown

Native HTML `<select>` dropdowns, available on a page object as
`modulars.dropdownActions`.

- [SelectDropdownActions](SelectDropdownActions.md) — select/deselect options, read selections.
- [DropdownActionException](DropdownActionException.md) — thrown when a dropdown action can't be carried out.
