# modulars/mouse

Mouse gestures — hover, drag and drop, scrolling — available on a page
object as `modulars.mouseActions`.

- [MouseActions](MouseActions.md) — the full set of mouse gestures.
- [MouseActionException](MouseActionException.md) — thrown when an element doesn't reach the needed state, or a gesture is misconfigured.
