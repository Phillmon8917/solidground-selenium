# modulars/calendar

Driving date-picker calendar widgets and representing calendar dates,
available on a page object as `modulars.calendarActions`.

- [CalendarActions](CalendarActions.md) — clicks through a calendar widget to select a date.
- [CalendarLocators](CalendarLocators.md) — describes where the widget's buttons/caption/days are.
- [DateObject](DateObject.md) — a validated calendar date with parsing and formatting.
- [DateFormat](DateFormat.md) — common date pattern presets used by `DateObject`.
- [DateOffset](DateOffset.md) — builds a `DateObject` relative to today.
- [CalendarNavigationException](CalendarNavigationException.md) — thrown when the calendar can't be navigated.
- [InvalidCalendarConfigurationException](InvalidCalendarConfigurationException.md) — thrown when `CalendarLocators` is built incompletely.
- [InvalidDateException](InvalidDateException.md) — thrown when `DateObject` is given a bad date.
