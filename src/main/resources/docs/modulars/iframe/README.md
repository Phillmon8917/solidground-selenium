# modulars/iframe

Switching into and out of iframes, available on a page object as
`modulars.iframeActions`.

- [IframeActions](IframeActions.md) — switch into/out of frames, track nesting depth, run an action inside a frame.
- [IframeActionException](IframeActionException.md) — thrown when a frame switch fails.
