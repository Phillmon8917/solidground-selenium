# modulars/keyboard

Keyboard input and shortcuts, available on a page object as
`modulars.keyboardActions`.

- [KeyboardActions](KeyboardActions.md) — single keys, chords, shortcuts, arrow navigation.
- [KeyboardActionException](KeyboardActionException.md) — thrown when the target element doesn't appear in time.
