# modulars/toggle

Switch/checkbox style toggle elements, available on a page object as
`modulars.toggleActions`.

- [ToggleActions](ToggleActions.md) — ensure/assert/read a toggle's on/off state.
- [ToggleStateResolver](ToggleStateResolver.md) — describes how to tell whether a toggle is on.
- [ToggleActionException](ToggleActionException.md) — thrown when the expected state can't be confirmed.
