# modulars

Every action group a page object can use, each exposed as a field on
[ActionsContainer](../base/ActionsContainer.md) (`modulars.xxxActions`).

- [assertions](assertions/README.md) — TestNG assertions with logging built in.
- [browser](browser/README.md) — whole-browser actions: navigation, windows, tabs, screenshots.
- [calendar](calendar/README.md) — driving date-picker calendar widgets and representing dates.
- [dropdown](dropdown/README.md) — native HTML `<select>` dropdowns.
- [element](element/README.md) — the main element actions: click, type, read text/attributes, wait.
- [iframe](iframe/README.md) — switching into and out of iframes.
- [keyboard](keyboard/README.md) — keyboard input and shortcuts.
- [loading](loading/README.md) — waiting for spinners, skeletons, and page loaders to disappear.
- [mouse](mouse/README.md) — mouse gestures: hover, drag and drop, scrolling.
- [network](network/README.md) — verifying real network requests and responses via Chrome DevTools.
- [reader](reader/README.md) — downloading and reading Excel, PDF, text, and Word documents.
- [toggle](toggle/README.md) — switch/checkbox style toggle elements.
- [wait](wait/README.md) — waiting for a value to stop changing (stabilize).
