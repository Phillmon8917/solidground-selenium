# modulars/wait

Waiting for a value to stop changing (stabilize), available on a page
object as `modulars.waitActions`.

- [SmartWaitActions](SmartWaitActions.md) — the stabilization wait methods.
- [WaitStabilizationException](WaitStabilizationException.md) — thrown when a value never settles in time.
