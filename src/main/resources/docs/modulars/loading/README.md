# modulars/loading

Waiting for spinners, skeletons, and page loaders to disappear, available
on a page object as `modulars.loadingIndicatorActions`.

- [LoadingIndicatorActions](LoadingIndicatorActions.md) — the wait methods themselves.
- [LoadingIndicatorException](LoadingIndicatorException.md) — thrown when an indicator doesn't disappear in time.
